# Contracts

Contract Management System for Frappe

## Features

- Manage contracts with comprehensive tracking
- Contract lifecycle management
- Automated notifications and reminders
- Document management
- Reporting and analytics

## Installation

1. Get the app:
```bash
bench get-app contracts
```

2. Install on site:
```bash
bench --site [sitename] install-app contracts
```

## Configuration

After installation, configure the app from:
Setup > Contracts Settings
